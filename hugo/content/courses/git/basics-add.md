Add a entire working directory to the staging area:

{{< file "terminal" "command line" >}}
```bash
git add .
```

Add a single file:

{{< file "terminal" "command line" >}}
```bash
git add somefile.txt
```

Remove a file from the staging area:

{{< file "terminal" "command line" >}}
```bash
git reset .
```
