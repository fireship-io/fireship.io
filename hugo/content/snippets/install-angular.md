Angular requires [Node version 8.x or 10.x installed](/snippets/install-nodejs/) installed on your machine.

## Quick Installation Steps

{{< file "terminal" "command line" >}}
{{< highlight terminal >}}
npm install -g @angular/cli

# create an app
ng new myApp

# serve it
ng serve -o
{{< /highlight >}}
